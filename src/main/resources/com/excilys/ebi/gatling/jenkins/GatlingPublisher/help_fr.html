Ce plugin permet d'intégrer le suivit d'une simulation Gatling à votre job Jenkins :

  • Suivi des performances au cours du temps
  • Publication des rapports

NB : Les résultats des simulations Gatling sont stockés dans le répertoire ${GATLING_HOME}/results/. Pour chaque run d'une simulation un dossier est créé selon le schéma suivant : mysimulation-20120911161003, la première partie étant généralement le nom de la simulation et la deuxième partie un timestamp. Il faut donc remplir le champ "Préfixe des répertoires de sortie" avec la première partie du nom de ce répertoire.
